# Saving SuperStore Analysis

## Overview
This project analyzes the Superstore dataset to identify profit centers, loss-making areas, advertising opportunities, and return patterns. The goal is to provide data-driven recommendations to improve business performance.

## Key Findings

### Profit & Loss
- Copiers and Chairs are among the strongest profit drivers.
- Tables and Binders are major loss-makers.

### Advertising
- Certain state-month combinations show higher profitability.
- Advertising investment should be up to 20% of profits (1/5 rule).

### Returns
- High-profit categories like Copiers should be maintained.
- Low-profit, high-return categories like Tables should be reduced.
